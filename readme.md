# 🍽️ Recipedia

**Recipedia** is a React Native mobile app that lets users explore food recipes categorized by global regions. Whether you're craving Italian pasta, Asian stir-fry, or Middle Eastern delights, Recipedia serves up curated recipes with intuitive navigation and a clean UI.

---

## 📱 Features

- 🌍 Region-based recipe categories (e.g., Italian, Asian, Mexican, etc.)
- 📖 Detailed recipe screens with ingredients and instructions
- 🔍 Easy navigation and filtering
- 💡 Responsive design for both Android and iOS
- ⚡ Fast performance with optimized builds

---

## 🚀 Getting Started

### Prerequisites

- Node.js ≥ 16.x
- React Native CLI or Expo CLI
- Android Studio / Xcode (for emulator or device testing)

### Installation and Steps to Run

```bash
git clone https://github.com/yourusername/recipedia.git
cd recipedia
npm install
npx react-native start
npx react-native run-android
npx react-native run-ios(for mac)
